{% comment %} This generic template renders a quick help tooltiped button Parameters: title : Title of the tooltip content : Content of the tooltip id : Optionnal. ID of the generated button hide : Optionnal. Hides the button if set, otherwise the button is visible by default. Example: {% include 'main/generic/help_button.html' with title='Some tooltip title' content='Some tooltip content.' %} {% endcomment %}
